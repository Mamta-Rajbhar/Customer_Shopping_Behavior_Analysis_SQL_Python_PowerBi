# Customer_Shopping_Behavior_Analysis_SQL_Python_PowerBi

🛍️ Customer Shopping Behavior Analysis
📌 Overview

Analyzes 3,900 customer transactions to identify spending patterns, customer segments, and product preferences for better business decisions.

📊 Dataset
3,900 records, 18 columns
Includes demographics, purchase details, and shopping behavior
37 missing values in review ratings (handled)

⚙️ Process
Python: Data cleaning, preprocessing, feature engineering
SQL: Business insights (revenue, top products, segmentation)
Power BI: Interactive dashboard for visualization

💡 Outcome
Improves marketing strategies, customer retention, and product positioning.

🚀 Tech Stack
Python | MySQL | Power BI
